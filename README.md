# digital.clock
check README for description

This is a digital clock which I have created using Python. 

I used tkinter and strftime to create the digital clock. 

The software I used was VS Code

The clock shows the time in hours, minutes, and seconds. 

It also shows the current date. 

Running the program in your terminal will display the clock on your home screen.
